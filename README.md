# virtual-assistant
Voice Controlled Virtual Assistant is a web application built using HTML, CSS, and JavaScript that recognizes voice commands and opens websites or browser applications like Google, YouTube, and Facebook using the Web Speech API.

# Voice Controlled Virtual Assistant Project Description

A Voice Controlled Virtual Assistant developed using HTML, CSS, and JavaScript that performs tasks based on user voice commands. The assistant listens to spoken instructions and automatically opens websites, browser tabs, or web applications such as Google, YouTube, Facebook, and Calculator.

The project uses the Web Speech API for speech recognition and JavaScript for processing commands and executing actions in real time. It provides a simple, responsive, and interactive interface that allows users to control basic browser operations hands-free.

## Features

* Voice command recognition
* Opens websites using voice input
* Launches browser-based applications
* Speech-to-text conversion
* Responsive and modern UI
* Real-time command processing

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Web Speech API

## Objective

The objective of this project is to create a smart browser-based virtual assistant that improves user convenience by enabling voice-controlled web navigation and quick access to online platforms.

## Example Commands

* “Open YouTube”
* “Open Google”
* “Open Calculator”
* “Open Facebook”
* “Search JavaScript tutorials”
